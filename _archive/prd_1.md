# ⚽ crazy-match - 世界杯比赛预测工具

## 1. 产品概述

### 1.1 产品名称
**crazy-match**

### 1.2 一句话描述
一款基于历史战绩和球员实力分析的世界杯比赛预测工具，帮助用户预测比赛结果并提供各队实力评分。

### 1.3 目标用户
- 足球爱好者
- 世界杯观众
- 彩票/竞猜参与者
- 数据分析爱好者

### 1.4 核心价值
- **历史战绩分析**：基于各队过往世界杯表现进行预测
- **球员实力评估**：根据球队阵容计算 100 分制实力评分
- **比赛预测**：结合历史 + 实力给出胜平负预测
- **数据可视化**：清晰展示各队实力对比

### 1.5 长期定位
- **V1（当前）**：2026 世界杯比赛预测
- **V2**：扩展到欧冠、欧洲杯等赛事
- **V3**：加入 AI 比分预测模型

### 2.2 预测模型

#### 目标一：基于历史战绩预测
通过以下维度计算历史得分：
- 过往世界杯总场次
- 胜/平/负场次
- 进球数/失球数
- 最好成绩（冠军/亚军/四强等）
- 最近三届表现（加权）

#### 目标二：基于球员实力评分（100分制）
计算维度：
- FIFA 排名
- 五大联赛球员数量
- 主力球员身价加成
- 主教练因素
- 阵容深度（替补实力）

综合公式：
```
球队实力分 = 历史分 × 40% + 球员分 × 60%
```

### 4.7 预测算法

```javascript
// 计算预测结果
function calculatePrediction(homeTeam, awayTeam) {
  // 历史分 (满分 40)
  const historyScore = calculateHistoryScore(homeTeam) * 0.4;
  
  // 球员分 (满分 60)
  const playerScore = calculatePlayerScore(homeTeam) * 0.6;
  
  // 综合实力分
  const homePower = historyScore + playerScore;
  
  // 同样计算客队
  const awayHistoryScore = calculateHistoryScore(awayTeam) * 0.4;
  const awayPlayerScore = calculatePlayerScore(awayTeam) * 0.6;
  const awayPower = awayHistoryScore + awayPlayerScore;
  
  // 计算胜率
  const powerDiff = homePower - awayPower;
  const homeWinProb = 1 / (1 + Math.pow(10, -powerDiff / 3));
  const awayWinProb = 1 - homeWinProb;
  
  return {
    home_win: homeWinProb,
    draw: 0.15, // 平局假设
    away_win: awayWinProb
  };
}
```

## 5. 实力评分标准

### 5.1 历史分计算（满分 40 分）

| 指标 | 权重 | 说明 |
|------|------|------|
| 冠军次数 | 15分 | 每冠 2.5 分，上限 15 |
| 参赛次数 | 5分 | 参赛超过 15 次得满分 |
| 胜率 | 10分 | 胜率 >60% 得满分 |
| 最近表现 | 10分 | 近三届成绩加权 |

### 5.2 球员分计算（满分 60 分）

| 指标 | 权重 | 说明 |
|------|------|------|
| FIFA 排名 | 15分 | 排名 1 得 15 分，递减 |
| 五大联赛球员 | 15分 | 每名主力加 1-3 分 |
| 身价总和 | 15分 | 身价前三球员加权 |
| 主教练 | 5分 | 名帅加成 |
| 阵容深度 | 10分 | 替补席实力 |
